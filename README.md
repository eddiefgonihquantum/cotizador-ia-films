# cotizador-ia-films

> Cotizador de presupuestos para produccion de films con IA — IA-FILMS x BangAI

## Descripcion

Herramienta web para cotizar presupuestos de produccion cinematografica con asistencia de inteligencia artificial. Desarrollada como parte del proyecto IA-FILMS en colaboracion con BangAI.

## Tecnologias

- HTML5
- CSS3 (estilos inline)
- JavaScript vanilla (sin frameworks)
- Desplegado en Vercel

## Uso

Abre el archivo `index.html` en tu navegador o accede al deploy en:

[cotizador-ia-films.vercel.app](https://cotizador-ia-films.vercel.app)

## Instalacion local

```bash
npx serve .
```

## Deploy

El proyecto se despliega automaticamente en Vercel al hacer push a la rama `main`.
